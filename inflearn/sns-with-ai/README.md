# SNS With AI

인프런 강의 [AI 다루는 백엔드 설계 기본 - SpringBoot SNS](https://www.inflearn.com/course/ai-%EB%8B%A4%EB%A3%A8%EB%8A%94-%EB%B0%B1%EC%97%94%EB%93%9C-%EC%84%A4%EA%B3%84-%EA%B8%B0%EB%B3%B8-spr)를 보며 만드는 SNS 백엔드 프로젝트입니다.

이 프로젝트에서는 단순히 기능 구현만 따라가기보다, AI Agent를 활용해 서비스 요구사항을 정리하고 백엔드 구조를 설계하는 흐름까지 함께 연습하는 것을 목표로 합니다.

## 학습 목표

- Spring Boot 기반으로 SNS 서비스의 핵심 기능을 직접 설계하고 구현한다.
- 사용자 인증과 세션, 팔로우, 게시글, 미디어, 타임라인 구조를 단계적으로 익힌다.
- AI Agent를 개발 보조 도구가 아니라 설계와 구현을 함께 진행하는 파트너처럼 활용해본다.

## 구현 예정 범위

- 사용자 회원가입 / 로그인
- 세션 기반 인증과 공유 세션
- 팔로우 기능
- 게시글 작성 / 조회
- 미디어 업로드
- 타임라인 구성

## 현재 기술 스택

- Java
- Spring Boot
- Spring Web MVC
- Spring Data JPA
- H2
- MySQL
